Instructions

Getting Started with gathurUP

Step 1: Add People

Start in the People and Groups tab:

  • Add yourself first
  • Add friends you plan to invite to events
  • Use full names for easier identification

Step 2: Create Groups

In People and Groups, organize your contacts:

  • Create different groups for different purposes
  • Family events, friend gatherings, trip groups, etc.
  • Add people to multiple groups as needed

Step 3: Create Events

Use Create Event to set up your gathering:

  • Single-Day Events:
    • Perfect for meetups, dinners, game nights
    • Option to add specific time slots
    • Multiple time options for flexibility
  • Multi-Day Events:
    • Ideal for trips and longer gatherings
    • Select date ranges for availability
    • Perfect for weekend trips or vacations
  • Location Options:
    • Add multiple location choices
    • Include descriptions and images
    • Let group vote on preferences

Frequently Asked Questions

Q: Do my friends need accounts to respond?

A: No! They just need the event link you share with them.

Q: How do date votes work?

A: Members use thumbs up/down to indicate availability for each date option.

Q: How do location votes work?

A: Members select their single preferred location from the options provided.

Q: Can I add times to multi-day events?

A: No, time slots are only available for single-day events.

Q: How are different timezones handled?

A: Times are automatically converted to each member's local timezone.

Q: Can I edit an event after creating it?

A: Yes! You can edit details, add/remove dates, and update locations anytime.

Q: What's coming next?

A: Hotel preferences, budget planning, activity planning, packing lists, and task assignments.
